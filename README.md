# Yahtzee 小程序

这是一个简单的骰子得分游戏微信小程序，基于 Yahtzee 惯例玩法。玩家轮流掷骰，选择得分项并确认，最后比较总分。

## 游戏目标

- 每位玩家轮流投掷 5 个骰子
- 根据骰子点数选择一个可得分的项目
- 先选择后确认得分，确认后切换到下一位玩家
- 每轮结束后更新轮次，最后统计玩家总分

## 基本玩法

1. 点击“投掷骰子”按钮，系统会掷出 5 个随机骰子。
2. 当前玩家可以在得分表中选择一个项目进行得分：
   - 一点到六点：统计当前骰子中相同点数的总点数
   - 全选：骰子点数总和
   - 四骰同花：四个相同点数
   - 葫芦：3+2 组合
   - 小顺：连续 4 个点数
   - 大顺：连续 5 个点数
   - 快艇：5 个相同点数
3. 点击一个得分项目之后，当前选中项会高亮并进入待确认状态。
4. 再次点击同一个项目即可确认得分，并自动切换到下一位玩家。
5. 确认后的得分格会保留高亮背景，表示该项目已经使用过。
6. 多轮结束后，玩家的总分会自动计算。

## 界面说明

- 当前玩家所在列会高亮显示。
- 当前轮次显示在表格顶部：`第 X 轮 / 12`。
- 投骰按钮在骰子区，掷骰中会显示动画。
- 页面底部有“重置”按钮，可清空所有分数并重新开始。

## 玩法要点

- 每位玩家只能在自己的列选择得分项。
- 选项确认前可以更换选择，确认后即固定该项目得分。
- 不同项目的得分规则不同，需根据当前骰子结果合理选择。
- 当轮结束后，轮数会增加。

## 运行方式

将此项目导入微信小程序开发者工具，打开 `pages/index` 页面即可体验游戏。
